Ejercicio 01 · ¿Qué es cliente-servidor?

Tipo: Proyecto individual

Instrucciones:

Relaciona cada elemento con su función en el modelo cliente-servidor.

Elementos

  1. Cliente
  2. Servidor
  3. Petición
  4. Respuesta

Funciones

  1. Envía datos solicitados al cliente.
  2. Solicita información o servicios.
  3. Acción que inicia la comunicación.
  4. Resultado que recibe el cliente.

Solución:

  • 1 - B: El cliente es el que solicita información o servicios.
  • 2 - A: El servidor envía los datos solicitados al cliente.
  • 3 - C: La petición es la acción que inicia la comunicación.
  • 4 - D: La respuesta es el resultado que recibe el cliente.
